# poliver.dev – Personal Portfolio

Source code for **poliver.dev**, a personal portfolio site for Patrick Oliver — software engineer specialising in technical direction and hands-on delivery for early-stage teams.

## Overview

- **Tech Stack**
  - **Frontend**: React + Vite
  - **Language**: TypeScript
  - **Styling**: Tailwind CSS + shadcn/ui
  - **Animation**: Framer Motion

- **Production URL**: `https://oxford-systems-builder.lovable.app`

## Site Structure

- **Hero** — Positioning statement and primary CTAs (Get in Touch, Download CV)
- **Who I Work With** — Types of teams and founders I collaborate with
- **About** — Background, approach, and the kinds of problems I solve
- **What I Bring** — Capabilities grouped by outcome: technical direction, full-stack delivery, applied AI, infrastructure
- **Experience** — Domain background across climate tech, biotech, and early-stage startups
- **Contact** — Contact form plus direct email link

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```
